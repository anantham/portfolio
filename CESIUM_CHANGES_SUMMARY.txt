=== INSTALLATION STEPS ===

On your Mac:

1. Install dependencies:
   npm install cesium resium copy-webpack-plugin

2. Copy all the new files from this cloud environment

3. Apply modifications to existing files

4. Commit and push:
   git add -A
   git commit -m "feat: Cesium.js 3D journey visualization"
   git push origin claude/redesign-interactive-website-01EajJQZujpgHpeiUx8jrMm5
